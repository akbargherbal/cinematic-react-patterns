‚ö†Ô∏è Common Mistakes

‚ùå Don't Do:

  • Using Redux for theme toggle (overkill!)
  • Context for everything (performance issues)
  • Props through 6+ levels (maintenance nightmare)
  • Multiple Contexts when one would work

‚úÖ Do Instead:

  • Start simple, upgrade when needed
  • Separate Contexts for separate concerns
  • Measure before optimizing
  • Document your choice for team

3. Zustand Pattern (Recommended Library)

Step 1: Create Store

import { create } from 'zustand';

const useCartStore = create((set) => ({
  items: [],
  
  addItem: (item) => set((state) => ({
    items: [...state.items, item]
  })),
  
  removeItem: (id) => set((state) => ({
    items: state.items.filter(i => i.id !== id)
  })),
  
  clearCart: () => set({ items: [] })
}));
